# Изображения CAREL Professional Service

Статика лежит в `public/images/`. В коде пути начинаются с `/images/…`.

## Основные папки

| Путь | Назначение |
|------|------------|
| `header/` | Логотип, иконки шапки |
| `features/` | Иконки блока «Преимущества» на главной |
| `parts-accessories/` | Карточки «Комплектующие» на главной |
| `parts.png` | Общее фото запчастей (`lib/partsImage.ts`) |
| `catalog/humisteam/` | Галерея и фото humiSteam — см. `catalog/humisteam/README.md` |
| `catalog/components/` | Запчасти — см. `catalog/components/README.md` |

Подробные инструкции по humiSteam и комплектующим — в README соответствующих подпапок `catalog/`.
